# unique-scents-website-part3
Project Overview

This project is a fully interactive website developed as part of Part 3 of the web development assessment. The goal of this stage is to enhance a previously created static website by adding JavaScript-driven interactive features, improving user experience, design, functionality, and overall professionalism.

The website includes features such as animations, modals, product filtering, image lightboxes, accordions, and form validation.
